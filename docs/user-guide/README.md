# Orbflow User Guide

Welcome to Orbflow — a visual workflow automation engine. This guide walks you through every page of the application with detailed descriptions and screenshots. Whether you are building your first workflow or managing a team of automations, start with any page below to learn how it works.

---

## Pages

| Page | Description |
|------|-------------|
| [Builder](builder.md) | Create workflows by adding nodes, connecting them, and configuring each step |
| [Activity](activity.md) | Monitor workflow runs, inspect execution details, and audit trails |
| [Analytics](analytics.md) | View execution metrics, latency percentiles, and per-node performance |
| [Templates](templates.md) | Browse pre-built workflow templates to get started quickly |
| [Credentials](credentials.md) | Manage encrypted secrets and API keys used by workflow nodes |
| [Marketplace](marketplace.md) | Browse, install, and manage plugins and integrations |
| [Reviews](reviews.md) | Propose, review, and approve workflow changes collaboratively |
| [Budgets](budgets.md) | Set spending limits and track execution costs |
| [Alerts](alerts.md) | Monitor workflow metrics and get notified when thresholds are breached |
| [Access Control](access.md) | Manage roles, permissions, and user bindings |

---

## Navigation

Orbflow is a single-page application. All pages are accessible from the sidebar on the left. Click any tab to switch views — your state is preserved across tabs.

The sidebar also shows:
- **Engine status** — green dot when the backend is online
- **Version** — current Orbflow version
- **Theme toggle** — switch between dark and light mode
